===============    
QInfer Examples
===============

We are currently in the process of converting examples from being Python modules
in the ``qinfer.examples`` package to `IPython Notebook`_ documents describing
the concepts behind QInfer. 

Notebook-Based Examples
-----------------------

If you have already installed `IPython Notebook`_, you can run the new examples
from your system command line, by navigating to where you downloaded
**QInfer**, then running::

    cd python-qinfer/examples
    ipython notebook

Alternatively, IPython provides an online viewer for reading notebook files:

- `Plotting Example <http://nbviewer.ipython.org/github/csferrie/python-qinfer/blob/master/examples/plot_example.ipynb>`_
- `Noisy Coin Example <http://nbviewer.ipython.org/github/csferrie/python-qinfer/blob/master/examples/Noisy%20Coin%20Example.ipynb>`_
